Команды для игроков Команды для игроков

в приват боту
Персонаж
!reg -Регистрация персонажа
!pass -Установить пароль
!ident -Идентифицироваться
!sex -Установить пол
!rebirth -Реморт (перерождение в нового) (подробнее тут )
!drop -Удалить персонаж
!hark -Распределить свободную характеристику (free hark)
Команда (team)
!com -Установить команду по умолчанию
!coms -Доступные команды
!comset -Действия с командой (подробнее тут )
Игровые моды
!mode -Cписок модов
!mode info -Описание мода
!mode + -Включает мод
!mode - -Выключает мод
Характеристики и вещи
!stat -Основные характеристики
!xstat -Дополнительные характеристики
!collections -Свои коллекции
!eq -Список вещей
!info -Одновременно !stat !xstat !collections !eq
!places -Посмотреть что надето
!shop -Магазин вещей
!buy -Купить вещь
!sell -Продать вещь (вещи из магазина продаются за 60% от стоимости, собственноручно сделанные вещи - за 30% от стоимости)
!ex -Характеристики вещи
!putoff -Снять вещь
!puton -Одеть вещь
!w -Активировать оружие
!throw -Выбросить вещь
!cex -Список коллекций
!cex код -Описание коллекции
!trans -Передать gold
Добыча ресурсов
!rmake -Добыть ресурс (требуется навык изготовитель)
!rex -Необходимый материал для добычи ресурса
!ttransfer -Передать добытый ресурс
Изготовление вещей
!tmake -Сделать вещь (требуется навык кузнец или ювелир)
!tex -Необходимый материал для изготовления вещи
!raw -Список материалов
!ttransfer -Передать собственноручно сделанную вещь
Изготовление зелий
!make -Сделать зелье
!poi -Список доступных зелий
!ptransfer -Передать собственноручно сделанное зелье
Татуировки
!tatoo list -Список доступных татуировок
!tatoo ex -Описание татуировки
!tatoo set -Активировать татуировку
Комплекты
!set save -Записать текущий комплект
!set list -Посмотреть список записанных комплектов
!set clear -Удалить комплект
!set put -Одеть комплект
Аукцион
!auction add -Начать аукцион
!auction stop -Остановить аукцион
!auction bid -Сделать ставку в аукционе
Умения (skill)
!slist -Список скиллов
!slearn -Выучить скилл
!skillex -Описание скилла
Навыки (ability)
!nlist -Список навыков (!nlist all – список всех навыков)
!nlearn -Выучить навык
!nex -Описание навыка
!resist -Выучить резисты (только для воинов и лучников)
Магия
!mag -Список доступной магии
!learn -Выучить магию
!mex -Описание магии
!aura -Установить длительную магию как ауру
Общение через бота
!say -Сказать в глобал
!me -Сказать в глобал от третьего лица
!msg -Сказать другому персонажу
!comsay -Сказать членам своей команды
!clansay -Сказать членам своего клана
Новости и письма
!readnews -Читать новость (если есть)
!letread -Читать письма (если есть)
!letnew -Написать письмо
Прочие
!settitle -Устаносить титул (приписка перед ником)
!nostat -Скинуть характеристики (стоимость 1000 gold)
!ver -Посмотреть версию игры
!killdcc -"Убить" дцц-коннект
!clook -Описание зарегистрированной команды (клана)
Во время боя
!do -Сделать заказ на бой (подробнее тут )
!clear -Очистить заказ
!mana -Количество маны
!ener -Количество энергии
!run -Сбежать из боя

!ptop craft -ТОП 5 игроков по изготовлению
!ptop miss -ТОП 5 игроков по промахам
!ptop suic -ТОП 5 игроков по промахам
!reg -Регистрация в игру
!unreg -Отмена регистрации в игру
!name -Имя команды
!com -Список команд
!list -Список игроков
!move -Кто не сделал заказ
!round -Время до конца раунда
!clook -Описание команды
!look -Описание игрока
